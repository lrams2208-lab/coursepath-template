# CoursePath

CoursePath is a web app that gives step-by-step guidance for school and work problems.

## Features
- Categories: School & Work
- Subcategories like Math, Science, HR, Marketing, etc.
- AI generates step-by-step instructions
- Colorful and lively interface
- Fully web-based, no local setup
